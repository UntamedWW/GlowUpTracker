# 🌟 Glow Up Tracker

**Glow Up Tracker** is a self-improvement Android app built with **Kotlin** and **Jetpack Compose**.

The app helps users build discipline through daily habits, weekly progress tracking, and motivational reinforcement.

---

## 📱 Features (MVP)

- Add custom habits (Workout, Reading, Coding, etc.)
- Mark habits as completed
- Weekly progress indicator
- Daily motivational message
- Streak tracking system
- Modern Compose UI

---

## 🧱 Tech Stack

- **Kotlin**
- **Jetpack Compose**
- **Material 3**
- **Navigation Compose**
- **Room** (planned for data persistence)

---

## 🎯 Project Goal

This project is built as a **learning-focused production-style application**.

Main goals:

- Practice modern **Compose UI development**
- Implement local persistence using **Room**
- Manage UI state properly
- Structure a clean Android project
- Prepare for future **Play Store release**
